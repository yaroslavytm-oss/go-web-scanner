===========================================================
   Go Anti-Malware Engine - Portable Mode
===========================================================

Дякуємо за використання Go Anti-Malware Engine! 
Цей файл містить інструкції для роботи з портативною версією сканера.

---
ШВИДКИЙ СТАРТ
---

1. Запуск у режимі сканування (CLI):
   ./web-scanner --mode=scan --path=/шлях/до/директорії --threshold=50

   - Прапорці:
     --mode=scan     : режим сканування файлів.
     --path          : шлях до папки, яку перевіряємо.
     --threshold     : рівень небезпеки (Score), вище якого файл вважається вірусом.

2. Запуск у режимі Web UI:
   ./web-scanner --mode=ui --addr=0.0.0.0:9999

   Після запуску відкрийте браузер та перейдіть за адресою:
   http://<IP_ВАШОГО_СЕРВЕРА>:9999

---
КОНФІГУРАЦІЯ
---

Сканер використовує файл `signatures.json` для виявлення загроз.
Переконайтеся, що файл `signatures.json` знаходиться в тій же директорії, 
що і виконуваний файл `web-scanner`. Ви можете редагувати цей файл, 
додаючи власні регулярні вирази для пошуку нових типів вірусів.
